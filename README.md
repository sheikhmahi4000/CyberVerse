# 🌐 CyberVerse
Welcome to **CyberVerse**, an educational platform designed to provide insights into the world of cybersecurity.

## 🚀 Features
* **Interactive Dashboard:** View simulated cyber metrics.
* **Phishing Education:** Learn how to identify and prevent phishing attacks.
* **Toolkit:** Access various resources for security testing.
* **Career Guide:** Explore pathways into the cybersecurity industry.

## 🛠️ Built With
* HTML5
* CSS3
* JavaScript

## 📂 How to Run
Simply open `index.html` in any modern web browser, or visit the live link via GitHub Pages!
